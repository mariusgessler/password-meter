# Password-Meter

A tool with which you can take a password's strenght. Based on Dropbox' zxcvbn.

Take a look at the [live-version](https://mariusgessler.com).
 
## Technologies 

Built with: 

* _Vue.js_ - 2.6.11
* _zxcvbn_ - 4.4.2 
* Hosted with _Netlify_
